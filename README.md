# official-wccup-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容不针对任何特定域名或具体网站，仅作为通用资源存档使用。

## 项目简介

这是一个资源归档仓库，主要存放各类独立的 HTML 页面。页面内容可能涵盖多种用途，包括但不限于信息展示、工具界面、演示页面等。所有页面均为静态 HTML 文件，无需后端服务即可直接打开浏览。

## 目录说明

仓库根目录下按页面功能或主题划分文件夹，每个文件夹内包含对应的 HTML 文件及其所需的资源（如 CSS、JavaScript、图片等）。目录结构如下：

```
official-wccup-resource/
├── page-group-a/       # 主题 A 相关页面
├── page-group-b/       # 主题 B 相关页面
└── ...
```

具体每个文件夹的用途请参考该文件夹内的 `README.md` 或文件注释。

## 页面归档说明

- **归档范围**：本仓库收录的 HTML 页面均为独立、完整的静态页面，不依赖外部动态接口或数据库。
- **页面状态**：部分页面可能为历史版本或实验性内容，请以实际使用时的效果为准。
- **使用方式**：可直接通过浏览器打开 HTML 文件，或部署至任意静态服务器进行访问。
